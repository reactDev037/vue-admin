## Prerequisites:

- [Node.js](https://nodejs.org/en/) (at least 4.x, 8.x recommended)
- npm version 3+ and [Git](https://git-scm.com/).

## Browser Support
* Latest Chrome, Firefox, Safari, Edge, IE11+

## Installation

``` bash
# create new project via vue-installer command.
$ vue myproject

or

# clone the repo
$ git clone https://github.com/reactDev937/vue-admin.git myproject

# go into app's directory and install dependencies:
$ cd myproject
$ npm install

# serve with hot reload at localhost:8080.
$ npm run dev

# build for production with minification
$ npm run build

# build for production and view the bundle analyzer report.
$ npm run build --report
```

* Vue.js
